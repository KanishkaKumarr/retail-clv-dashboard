# Retail Customer Retention & CLV Dashboard – Adidas

**Tools:** Power BI, DAX, Excel  
**Project Type:** Data Analysis / Dashboard  

## Objective
Analyze customer churn, repeat purchase behavior, and lifetime value (CLV) to improve retention strategies for Adidas.

## Steps
1. Imported and cleaned sales and customer data using Power BI and Excel  
2. Created interactive dashboards with DAX to track churn, repeat purchases, and CLV  
3. Segmented customers by region, channel, income, and loyalty tier  
4. Identified at-risk customers and recommended retention strategies  

## Outcome
Insights from the dashboard helped identify high-risk customer segments and informed targeted retention strategies.
